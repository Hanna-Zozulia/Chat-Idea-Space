# Chat Project

---

## Описание

Этот проект представляет собой простой чат с использованием **Express**, **Socket.io** и **TypeScript**, реализованный по принципам **MVC**.  
Поддерживает:  

- Отправку сообщений  
- Редактирование сообщений только своим автором  
- Удаление сообщений  
- Ответ на сообщения (reply)  
- Копирование текста сообщений  
- Онлайн-синхронизацию между всеми подключенными клиентами  

---

## Технологии

- Node.js  
- Express  
- Socket.io  
- TypeScript  
- ngrok (для публичного доступа к локальному серверу)  
- dotenv (для скрытия конфиденциальных данных, таких как ngrok authtoken)  

---

## Установка и запуск

1. Клонируйте репозиторий:

```bash
git clone 
```

2. Установите зависимости:

```bash
npm install
```

3. Создайте файл `.env` в корне проекта и добавьте ваш ngrok authtoken:

```bash
NGROK_AUTHTOKEN=ваш_токен_от_ngrok
```

4. Запуск в режиме разработки:

```bash
npm run dev
```

5. Запуск готового сервера:

```bash
npm run tunell
```
